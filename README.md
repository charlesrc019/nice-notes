# Nice Notes

Nice Notes is a simple web application designed to send uplifting, personalized messages to members within an organization. Its interface allows you to select recipients from a CSV roster and send SMS messages with ease. 

## Features

- Upload and manage a CSV list of members.
- Select one or multiple recipients with a searchable dropdown.
- Compose and send custom messages via SMS.
- Responsive design powered by Bootstrap.
- Real-time alerts for successful or failed message delivery.
